Using syze:
syze.ranges([an array of the sizes you would like to query on the browser window]);
syze.rangeNames({ an object of the sizes and the corresponding friendly names you would like to give them});
syze.threshold({ the number of milliseconds you would like to force between syze calls });

Keep it byte sized. Syze is 590 bytes.
